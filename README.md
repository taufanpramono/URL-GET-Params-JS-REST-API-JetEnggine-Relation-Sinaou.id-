# URL-GET-Params-JS-REST-API-JetEnggine-Relation-Sinaou.id-
code ini digunakan untuk project https://sinaou.id/ :
1. Mengambil data dari Post Type Title untuk post type nama Pengajar / guru 
2. Mengambil data dari REST API JET Enggine Relatiional Pengajar & Kelas
3. meneruskan data tersebut ke URL parameter /?namaguru=[shortcode_params]&kelas=[shortcode_params]
4. menangkap value parameter tersebut menggunakan javascrip
5. tampilkan data tersebut berdasarkan ID form field

**Bahasa :**
1. Native PHP
2. Java Scrip
